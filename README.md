# OpenCV Course Experiments

这是我的 OpenCV 学期课程实验归档。仓库保留了实验源码和主要实验结果图片，用于记录课程学习过程，也作为简历中的计算机视觉基础项目展示。

## 项目概览

本课程实验使用 C++ 和 OpenCV 完成，内容覆盖传统图像处理中的基础操作、相机标定、阈值分割、颜色分割、形态学处理、边缘检测、轮廓分析、图像插值和 Hough 直线检测。

通过这些实验，我掌握了从原始图像读取、预处理、特征提取到结果可视化保存的完整流程，并理解了不同图像处理算法在目标提取和几何特征检测中的适用场景。

## 实验内容

| 实验 | 主要内容 | 关键技术 |
| --- | --- | --- |
| 实验课1 | OpenCV 基础操作 | 图像读取、显示、保存、灰度化、滤波、边缘检测、缩放、绘图 |
| 实验课2 | 相机标定与去畸变 | 棋盘格角点检测、相机内参估计、畸变参数计算、图像去畸变 |

## 学习收获

### 1. OpenCV 基础工程能力

我掌握了 OpenCV 中常用的图像读写、窗口显示、矩阵操作和结果保存方式，能够使用 `Mat` 组织图像数据，并完成基本的图像处理流水线。

### 2. 图像预处理能力

我熟悉了灰度化、滤波、缩放、颜色空间转换等预处理操作，理解这些步骤对后续阈值分割、边缘检测和几何检测结果的影响。

### 3. 目标分割与形态学处理

我对比了固定阈值、Otsu 自动阈值和 HSV 颜色分割方法，并使用腐蚀、膨胀、开运算、闭运算优化二值图结果，减少噪声和孔洞问题。

### 4. 边缘检测与轮廓分析

我使用 Canny 和 Sobel 提取图像边缘，并通过 `findContours` 获取目标轮廓，理解了阈值、滤波核和形态学核大小对检测效果的影响。

### 5. 相机标定基础

我完成了基于棋盘格图像的相机标定实验，理解了相机内参、畸变参数和去畸变流程，为后续学习视觉测量、机器人视觉和三维视觉打下基础。

### 6. 几何特征检测

我使用 HoughLinesP 检测图像中的直线段，并根据长度、斜率和距离过滤无效线段，将像素级边缘信息进一步转化为结构化几何特征。
